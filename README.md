# Kali Linux Scripts Utils
Kali Linux Script Utils é um script de configuração e instalação para o Kali Linux, que busca cobrir configurações iniciais/intermediarias e instalação de pacotes que a distro não cobre por padrão.

**O script fornece ferramentas como:**
+ Instalação e configuração de Tor & Proxychains
+ Instalação e configuração do [Atuin](https://docs.atuin.sh/cli/guide/getting-started/)
+ (beta) Instalação de pacote de quebra de senhas + Wordlists
